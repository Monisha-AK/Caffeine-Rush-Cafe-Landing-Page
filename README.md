# Caffeine-Rush-Cafe-Landing-Page
A landing page for a coffee shop, built with HTML and CSS. Features a simple look and a basic cafe design.

A 5-page café website built with HTML, CSS, and JavaScript, showcasing the menu, story, and contact details of a fictional coffee shop.

## Pages
- `index.html` — Hero, brand pillars, signature picks, opening hours
- `menu.html` — Full coffee menu (12 drinks) with prices
- `coffee.html` — Detailed coffee guide with descriptions and intensity ratings
- `about.html` — Brand story, values, team, and awards
- `contact.html` — Contact form, business info, and embedded map

## Features
- Warm espresso/caramel themed design with glassmorphism cards
- Responsive hero with logo
- Working contact form with validation and toast confirmation
- Anchor-linked menu → coffee detail navigation
- Fully responsive layout

## Tech Stack
- HTML5
- CSS3 (custom properties, backdrop-filter, grid/flexbox)
